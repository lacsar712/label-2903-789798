NEV QUANTUM 是一个专为新能源汽车行业打造的高级数据可视化大屏系统。它结合了现代 Web 技术与工业级渲染引擎，提供从实时数据监控到深度业务分析的全栈解决方案。

---

## 🚀 核心功能与亮点

### 1. 深度多维分析 (Multi-dimensional Analysis)
*   **智能筛选与显性排序**：支持品牌、价格、续航、动力类型的复合过滤。
*   **实时更新联动**：所有筛选操作（下拉框、复选框、输入框）均支持**自动触发刷新**，无需手动点击。
*   **显性排序功能**：支持按**续航、价格、销量、电耗**进行升降序排列，图表与底部数据明细表同步更新。
*   **相关性挖掘**：提供“车身重量与电耗相关性”散点图，直观展现能耗效率分布。

### 2. 交互式地理信息 (Interactive GIS)
*   **双维度热力图**：支持“销量分布”与“充电桩密度”一键切换。
*   **性能黑科技**：引入 **GeoJSON 客户端缓存**技术，地图维度切换与筛选响应达到毫秒级，告别网络延迟。
*   **全参数映射**：地图维度现已完美兼容所有筛选条件（价格/续航/类型），实现真正的全屏联动。
*   **DataV 高德镜像**：集成阿里云 DataV 精准地图资源，支持滚轮无限缩放。

### 3. 企业级权限与账户管理 (Account Control)
*   **权限分级**：区分管理员 (Admin) 与普通用户。
*   **管理员特权**：支持对车型数据的 CRUD（增删改查）以及**用户账号管理**（新增、编辑、删除、重写权限）。
*   **安全中心**：支持用户**注册、登录、修改密码**，基于 Bcrypt 算法确保存储安全。

## 🛠 技术栈

| 层次 | 技术选型 | 备注 |
| :--- | :--- | :--- |
| **后端** | Python 3.9 + Flask | 轻量化、高并发、易扩展 |
| **持久层** | SQLAlchemy + SQLite | ORM 映射，支持平滑迁移至生产级数据库 |
| **认证管理** | Flask-Login + Bcrypt | 安全的会话管理与强加密密码哈希 |

| **可视化** | ECharts 5.5 | 行业顶尖渲染引擎，支持复杂交互联动 |
| **部署** | Docker + Compose | 环境隔离，一键部署 |

## 📦 快速部署

### 1. 环境准备
确保您的机器已安装 `Docker` 和 `Docker Compose`。

### 2. 启动服务
在项目根目录下执行：
```bash
docker-compose up --build -d
```

### 3. 访问与操作
*   **访问地址**：[http://localhost:12903](http://localhost:12903)
*   **管理员账号**：`admin`
*   **默认密码**：`admin123`
*   **第一步操作**：登录后点击右上角 **“重置模拟数据”** 以激活全量热力图展示与充电桩分布。
